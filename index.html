Introduction

CSS(Cascading Style Sheets); by definition; is a stylesheet language used to describe the presentation of a document written in markup languages such as HTML or XML. HTML/XML is the information(text, images, video, etc.) on the webpage while CSS is the design of the webpage. CSS Declaration

In order to change an HTML element; you need to make a CSS declaration on that specific element. CSS declarations consist of two things:

  • The property
    • Features you want to change about the HTML element.
    • Examples:
      • Font Size
      • Background Color
      • Width and Height of an Image
  • The value
    • Numerical or string value of how “much” you want to change that feature.
    • Numerical values paired with units of the developer’s choosing. More explained on units in CSS Units
  • Examples:
      • /* Element's background color is changed to red */ background-color: red; // Colon seperates the property and value
    • 26 would be the value is the font size in the HTML element
    • Red would be the value of the background color property of an HTML element

CSS Blocks

CSS declarations can be grouped together in a CSS block; which makes changing many aspects of an HTML element easier and makes code more organized. The example below groups two declarations together and changes all h1 tags on the page.

/* This CSS block changes the h1 element text color to be red and also changes its font style to be italic. */ h1 { color: red; font-style: italic; }

Inside the bracket and star combo characters /* */ is where comments/notes can be made about the CSS you are writing what its exactly doing.

CSS Selectors

There are six types of selectors that can change HTML elements and depending on their specificity can override other selectors changes to an element.The main ones are type, class, and id.

Type Selector

The type selector is where you select a specific element and change all its appearances on the page. The previous code is an example of how to use type selectors. All h1 on the page would be red and italized. Another example would be having all the heading 1 and paragraph tags have text that is green.

h1, p { color: green; }

CSS blocks can motify more than one element at time using a ",". This goes for class and id selectors as well. You can fix a CSS block with a combination of type, class, and id selectors.

Class Selector

The class selector is when you select an element based on the class attribute given to that element in the HTML code. For example, the HTML code below gives a paragraph tag a class attribute of red.

<p class="red">Hello</p>

In order to target the paragraph tags with class attribute "red" in CSS, you write CSS blocks such as this:

.red { color: red; }

The class selector is identified in CSS with "." before calling the class in the CSS block. Whatever element has the class attribute "red" will change the text in that element to be red. Class selectors are used when you want to change mutiple elements and give them the same changes, but don't want to change all of that element type.

ID Selector

The id selector is similar to the class selector as it selects an element based on the id attribute of that element. The code below shows how to assign an element as id attribute in HTML:

<button id="submit">Submit</button>

In order to target a tag with id attribute "submit" in CSS, you write CSS blocks such as this:

#submit { border: 4px solid blue; }

The id selector is identified in CSS with "#" before calling the id in the CSS block. Whatever element has the id attribute "submit" will change the border of that element to be a solid blue line that is 4 pixels wide.

Specificity

All selectors have a level of specificity and depending on that level, a selector can override the property another selector had already changed. Out of the main three selectors: type has the lowest, class is second, and id has the highest (the id selector has the highest specificity out of all selectors). To give an example of specificity; the code below shows three CSS blocks overriding the same element. Even though the type selector was used first; the text will be blue since the id selector has the highest specificity. Comment out the id CSS block and the text will be green since the class selector is the next selector on that element with the highest specificity being called on in the CSS code.

<p class="p1" id="p2">Hello there<p>
p { color: red; } .p1{ color: green; } #p2{ color: blue; }
